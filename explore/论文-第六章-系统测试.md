# 6 系统测试

为保证系统满足第三、四章所述功能与非功能需求，并在第五章实现基础上稳定运行，本章给出测试环境说明，采用黑盒方法抽取**代表性用例**（完整矩阵可置于附录）；并对核心算法做结果核对、对性能与安全做抽查。

## 6.1 测试环境

测试在与开发、部署一致的环境下进行，账号覆盖管理员、团队管理者与普通成员，团队中具备已批准成员与多状态任务，以便覆盖任务协作、绩效与看板等主要场景。具体软硬件配置见表 6.1。



**表 6.1　测试环境配置表**


| 项目       | 测试环境                                                                    |
| -------- | ----------------------------------------------------------------------- |
| 客户端硬件    | Intel Core i5 及以上 x64 处理器，16 GB 内存，固态硬盘                                 |
| 操作系统与浏览器 | Windows 10/11（64 位）；Chrome 或 Edge 稳定版                                   |
| 后端与数据库   | JDK 17（与 `backend/pom.xml` 一致），Maven；MySQL 8.x，参数与 `application.yml` 一致 |
| 前端构建     | Node.js LTS，Vite（与 `frontend/package.json` 约定一致）                        |




辅助工具以浏览器开发者工具为主，必要时使用 Postman、Apifox 等核对鉴权与状态码。
